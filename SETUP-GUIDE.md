## छवि तयारी निर्देशन

### गुणस्ता मापदण्ड:
- **रेजोलुसन:** उच्च गुणस्ता (कम्तिमा 72 DPI)
- **फाइल साइज:** प्रति छवि 500KB भन्दा कम (वेब गति को लागि)
- **फर्म्याट:** JPG (फोटो), PNG (लोगो/आइकन)
- **रङ:** चम्किलो, स्पष्ट, व्यावसायिक

### छवि अप्टिमाइजेसन:
यदि छविहरू धेरै ठूलो छन् भने, यी नि:शुल्क टूलहरू प्रयोग गर्नुहोस्:
- **TinyPNG** (https://tinypng.com/) - PNG/JPG compression
- **Squoosh** (https://squoosh.app/) - छवि compression र resize
- **ILoveIMG** (https://www.iloveimg.com/) - छवि resize र optimize

## फाइल राख्ने चरणहरू

### चरण १: फोल्डर तयार गर्नुहोस्
```
public/
├── images/
│   └── gallery/
└── sounds/
```

### चरण २: छविहरू प्राप्त गर्नुहोस्
- फोटोग्राफरबाट उच्च गुणस्ताका फोटो लिनुहोस्
- पुरानो कार्यक्रमका फोटो सङ्कलन गर्नुहोस्
- आवश्यक भए नयाँ फोटो खिच्नुहोस्

### चरण ३: छविहरू सम्पादन गर्नुहोस्
- सही आकारमा काट्नुहोस् (crop)
- चमक र कन्ट्रास्ट समायोजन गर्नुहोस्
- आवश्यक भए फिल्टर लगाउनुहोस्

### चरण ४: फाइलहरू राख्नुहोस्
- सही नाम र स्थानमा copy गर्नुहोस्
- सबै फाइलहरू जाँच गर्नुहोस्

### चरण ५: परीक्षण गर्नुहोस्
```bash
npm start
```
- वेबसाइट खोल्नुहोस्
- सबै छविहरू लोड भयो कि जाँच्नुहोस्
- घण्टी आवाज सुनिन्छ कि जाँच्नुहोस्

यदि केही छविहरू तत्काल उपलब्ध छैनन् भने:

- खोज्नुहोस्: "bell ring" वा "temple bell"

## समस्या समाधान

### छवि देखिएन भने:
1. फाइल नाम र स्थान जाँच गर्नुहोस्
2. फाइल नाम case-sensitive छ (ठूलो/सानो अक्षर)
3. फाइल एक्सटेन्सन सही छ कि जाँच्नुहोस् (.jpg, .png)
4. ब्राउजर refresh गर्नुहोस् (Ctrl + F5)

### आवाज बजेन भने:
1. फाइल स्थान जाँच गर्नुहोस्: `public/sounds/bell-ring.mp3`
2. MP3 फर्म्याट जाँच गर्नुहोस्
3. ब्राउजर console error हेर्नुहोस् (F12)
4. ब्राउजरमा sound permission दिनुहोस्
